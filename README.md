# p(ow)roxy
Proof of work proxy to deter web scraping

## Build
```bash
git clone https://github.com/B00TK1D/powroxy.git
cd powroxy
go build
```

## Usage
```bash
# ./proxy -h
Usage of ./powroxy:
  -port string
        port to listen on (default "8080")
  -url string
        target server (default "http://127.0.0.1:80")
```
